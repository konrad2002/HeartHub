# HeartHub - Project Idea

## Summary
HeartHub is a shared space for couples to collect notes, workouts, trips, wishlists, and recipes in one place. The entry point is a dashboard that shows recent activity and highlights what matters next.

## Goals
- Fast, low-friction capture of small moments (notes, memories, ideas).
- A unified timeline across categories.
- Simple, privacy-first sharing for two people.
- A product that can scale to more features without rebuilding.

## Core Features (MVP)
- Couples workspace with two members.
- Notes: quick text notes and pinned favorites.
- Training history: workout entries with date, duration, and tags.
- Places visited: location entries with date and notes.
- Places to visit: wishlist with priority.
- Recipes: title, ingredients, steps, and status (planned or cooked).
- Dashboard: recent items, upcoming plans, quick add.

## Optional Features (Phase 2)
- Photo attachments.
- Calendar view and reminders.
- Shared goals and milestones.
- Exports and backups.

## Users and Roles
- Two members per couple.
- Both can read and write by default.
- Optional: read-only guest later.

## Non-Goals (for now)
- Social network or discovery features.
- Public profiles.

## Success Criteria
- Time to create a new item is under 20 seconds.
- Dashboard shows the last 7 days and upcoming plans.
- User can find any item within 2 taps or 2 clicks.
